# Branch Documentation

Setiap anggota kelompok **wajib memiliki README** di branch masing-masing yang menjelaskan fitur/backlog yang dikerjakan.

## Template README untuk Branch Individu

Salin template di bawah ini ke file `README_BRANCH.md` di branch kalian masing-masing.

---

### Template:

```markdown
# Branch: [Nama Branch]

## Anggota
- **Nama**: [Nama Lengkap]
- **NPM**: [NPM]

## EPIC / PBI yang Dikerjakan

| PBI | Judul | Status |
|-----|-------|--------|
| PBI-X | [Judul PBI] | 🔲 Belum Mulai / 🔄 In Progress / ✅ Done |
| PBI-Y | [Judul PBI] | 🔲 / 🔄 / ✅ |

## Penjelasan Fitur

<!-- Jelaskan secara singkat fitur yang dikerjakan -->

## File yang Dimodifikasi / Ditambahkan

| File | Perubahan |
|------|-----------|
| `models/xxx.py` | Menambahkan model XXX |
| `views/xxx_views.xml` | Menambahkan form & tree view |

## Cara Test

<!-- Jelaskan cara test fitur secara manual -->

## Catatan

<!-- Catatan tambahan, dependency, blocker, dsb -->
```
